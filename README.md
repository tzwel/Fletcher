### A simple and fast solution to fetching data with JavaScript
Fletcher.js size - ~350 bytes unminified

## Usage

#### Check if Fletcher is working
```js
Fletcher("test");
```
Fletcher should log "Fletcher" in the browser's console if it's working.

#### Basic Fletcher function

```js
Fletcher("example.api/data/");
```
#### Assigning Fletcher to a variable:
```js
const data = Fletcher("example.api/data/");
```
